# User Guide

## Inventory Lookup
Use the lookup screen to search by LWH ID, customer, item, lot, bay, or other visible fields.

## Receiving / InvRec Print
Enter an InvRec value to find all associated pallets and print labels in bulk.

## Camera Scanner
Use the scan button on supported browsers/devices. Android Chrome/Edge works best.

## Printing
Use browser print preview to confirm orientation and scale before printing to Zebra.
