Distance / Route Planner

City+state or ZIP code per stop — e.g. "Fort Smith, AR" or "72901". Add stops for multi-point trips (A → B → C → back). Optional "Dwell" minutes per stop (loading, unloading, a break) get added into the day total below — drive time is calculated automatically, dwell time is whatever you enter since that varies by load. Uses free, keyless map services (OpenStreetMap + OSRM) — no account or API key needed, but as a free shared community service it can occasionally be slower or briefly unavailable, unlike a paid service with an uptime guarantee.
