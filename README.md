# LWH Warehouse Toolkit v1.2.3

Internal Logistics Warehouse PWA for lookup, labels, receiving print, signs, contact QR cards, and visitor badges.

## v1.2.3 focus
- One visible **Master Lookup** for the team.
- Master Lookup searches the customer/master CSV source.
- Receiving Print remains available for InvRec batch label printing.
- Camera scanner remains available in lookup/search workflows.

## Deployment
Upload the full unzipped contents to the GitHub Pages repository root and hard refresh after deployment.
